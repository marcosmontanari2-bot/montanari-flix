# montanari-flix
Montanari Flix é um projeto pessoal de frontend que desenvolvi para praticar e demonstrar meus estudos em HTML, CSS e JavaScript. É uma página inspirada em plataformas de streaming, focada na exibição de trailers de filmes, séries e desenhos. Criado para aplicar conceitos de layout responsivo e compartilhar minha evolução no LinkedIn.
